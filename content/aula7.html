Aula 7

Exemplo de Função

Function MinhaSoma (selecao As Range) As Integer
  soma = 0
  For Each celula In selecao
    soma = celula.Value + soma
  Next celula

  MinhaSoma = soma 'O resultado da função será a soma total'
End Function

Onde, dados, que será um Range de valores, é a entrada da função e Integer é o tipo do retorno. Precisamos obrigatoriamente informar estes dados.

Exercícios

  1. Faça uma função que calcule a média de um certo conjunto de dados.
  2. Faça um subprocedimento ou uma função que analise se certo número é triangular. Um número é triangular se ele for divisível por 2 E por 3. Exemplo: 6 = 2 x 3, 6 é triangular. 12 = 2 x 3 x 2, 12 é triangular. 5 não é triangular.

  3. Faça uma função que, dado um conjunto de dados, retorna a soma dos números pares.
  4. Faça uma função que, dado um conjunto de dados, retorna a soma dos números triangulares.
  5. Faça um subprocedimento que verifica se um número qualquer é primo.
  6. Faça uma função que, dado um conjunto de dados, retorna o menor valor encontrado.
  7. Faça uma função CalcMediaEMinimo que, dado um conjunto de dados, calcula a média e também acha o menor número na seleção. O resultado da função deve ser no formato "MEDIA # MINIMO". Por exemplo: se a média da seleção for 3,25 e o menor valor encontrado for 2, a função deverá retornar a String "3,25 # 2".
